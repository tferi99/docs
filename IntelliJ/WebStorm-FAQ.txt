Howto set single quote in TypeScript code?
==========================================
Auto import generates double quote by default. To change it to single quote:

	- Settings/Preferences
	- Editor | Code Style | TypeScript
	- "Punctuation" tab | Generated code -> Quote marks

SEE ALSO:	https://stackoverflow.com/a/39779498/783119

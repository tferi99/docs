========================
Working fast tricks
========================

Adding import in import section
-------------------------------
1. imp + Ctrl-Shift -> import
2. type start of the class, Ctrl-Shift (not start of import path!!!)


Adding import by class
----------------------
1. Typing a classname into (by typing start of the class, Ctrl-Shift)
2. import will be added automatically

Navigating to a method of a class
---------------------------------
1. Ctrl-Shift-H or Ctrl-Shift-T to type dialog
2. start typing classname
3. choose class
4. Ctrl-O to quick outline
5. start typing method name
6. choose method

Incremental find
----------------
1. Ctrl-J
2. Type text to find
3. Press Up/Down to find previous/next items.
